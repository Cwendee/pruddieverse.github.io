### Objective
To create a "Recruiter-Ready" digital presence that demonstrates a transition from Service Assurance to DevOps.

### Tech Stack
- **Framework:** Hugo
- **Styling:** Tailwind CSS
- **Deployment:** GitHub Pages
- **CI/CD:** GitHub Actions (Coming Soon)

### Key Achievements
- Configured custom Hugo layouts without relying on third-party theme constraints.
- Optimized asset delivery for high-performance lighthouse scores.
- Established a clean, professional "Midnight Navy" brand identity.
